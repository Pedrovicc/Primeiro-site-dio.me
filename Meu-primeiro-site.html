Tags Ensinadas

Voltar
  • Dentro das Tags temos a HTML que define a página ou seja, tudo que estiver entre ela vai ser considerado uma página HTML.
  • A Tag head pré-carrega antes da exibição para o usuário.
  • Tudo que estiver entre o body será oque de fato será mostrado para o usuário.
  • Tudo que estiver entre a Tag i será mostrado dentro do navegador em itálico.
  • Tudo que estiver entre a Tag strong será mostrado dentro do navegador em negrito.
  • Podendo assim unir, misturar, intercalar, as duas Tags para gerarmos um texto em negrito e itálico.
  • Boa parte das Tags precisam ser abertas e fechadas, porém existem algumas Tags que não necessitam ser fechadas, como por exemplo o imput.
  • A Tag img nos ajuda a colocarmos imagens tanto do nosso dispositivo quanto diretamente da internet.
  • Uma Tag muito útil é a Tag mark, que permite marcamos um texto.
  • Se você precisa riscar uma palavra ou texto, só usar a Tag s
  • Para sublinhar uma palavra basta usar a Tag u
  • Para colocar um subscrito basta colocar a Tag sup ex: 22

Atributos das Tags

Voltar
  • Existem atributos mais específicos e mais genéricos.
  • Dentro da Tag stronger podemos adicionar um atributo chamado id, que vai ajudar a identificar um elemento quando trabalhado em js, temos a tag style que vai poder ser usada junto com o css, outro atributo também voltado para o css seria o class.
  • A Tag imput possui um atributo type assim mudando a forma de formatação é resultado.
  • Um atributo usado no img é o src que vai direcionar um caminho de busca para a imagem, juto ao src podemos colocar o width para redimensionar a largura da imagem.

Listas ordenadas e não ordenadas

Voltar
  • As listas possuem um padrão no qual o navegador interpreta, podendo assim ser mudado via css.
  • No HTML possuímos listas ordenadas(sendo chamadas pelas Tags ol e li), e as listas não ordenadas(sendo chamadas pelas Tags ul e li).
  • As listas ordenas vão progredindo numericamente, em quanto as não ordenadas vão seguindo “parágrafos”.
